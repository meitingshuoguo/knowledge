# 重点概念

## 响应式系统（变化侦测）

1. 侦测一个对象的变化

   - `Object.defineProperty`
   - `Proxy`

2. 侦测一个数组的变化

   - 采用拦截器，将 7 个可以改变数组自身内容的方法进行覆盖

3. 收集依赖

   - 需要将对象的所有属性（包括子属性）都转成 getter/setter 的形式
   - 在 getter 中收集依赖，在 setter 中触发依赖。
   - 收集谁，就是当属性变化后，通知谁。就是`Watcher`
   - ES6 之前，只能侦测属性是否有修改，不能侦测属性被添加或删除。需要用`vm.$set`和`vm.$delete`
