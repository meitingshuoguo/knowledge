# Oh-My-Zsh Git

| 别名    | 全名                               |
| :------ | :--------------------------------- |
| `g`     | git                                |
| `ga`    | git add                            |
| `gb`    | git branch                         |
| `gcam`  | git commit -a -m                   |
| `gcan!` | git commit -v -a --no-edit --amend |
| `gcmsg` | git commit -m                      |
| `gco`   | git checkout                       |
| `gd`    | git diff                           |
| `gl`    | git pull                           |
| `gp`    | git push                           |
| `gm`    | git merge                          |
| `grb`   | git rebase                         |
